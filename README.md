#  Interactive Dragon Animation

A fun and interactive web experience featuring a colorful animated dragon that follows the user's mouse cursor.

The website allows users to interact with the dragon and customize its appearance and movement speed.

##  Features

-  Interactive animated dragon
-  Dragon follows the mouse cursor
-  Multiple color options
-  Adjustable animation speed
-  Color customization
-  Responsive interactive experience
-  Smooth animation
-  Fun and engaging user interaction
-  Works on desktop and mobile devices
